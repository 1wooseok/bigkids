### <b>소개</b>

어린이들을 위한 <strong>뉴스를 제공하는 웹사이트</strong> 입니다.

각종 시각화 도구를 사용해 오늘의 뉴스를 한눈에 볼 수 있습니다.

<br>

### <b>링크</b>

https://bigkids.co.kr

<br>

### <b>기간</b>

22.06.22 - 22.07.15

### <b>맡은 역할</b>

```
- 차트놀이 페이지 개발 ( 메인페이지 )

- 서버 데이터 요청 및 시각화

- 워드클라우드, 네트워크 그래프, 선 그래프, 테이블을 사용한 데이터 시각화

- 웹페이지 PDF 다운로드

- 반응형 웹페이지
```
